# DropGame

```
自由落下するゲームを作成する
```

## ビルド環境

- Python 3.11.0
- pip 22.3
- pyinstaller 5.8.0
- pygame 2.5.2
- pymunk 6.6.0

## ライブラリのインストール

```
$pip install pygame
```

## ビルド

- 下記コマンドを実行する

```
pyinstaller Main.py --noconsole --onefile --clean
```

## 運用対象

- Windows 10 以上
